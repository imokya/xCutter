[![xCutter logo](https://raw.githubusercontent.com/imokya/xcutter/master/renderer/img/icon.ico)] xCutter
一个基于Electron的桌面App，只需拖拖拽拽就能快速切图并生成HTML项目


### Features
* 支持图片压缩优化
* 支持监测代码(tracking)

### Usage
```
> npm i
> npm run dist
```
